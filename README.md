## Development

> `npm i`
> `npm prepare`

the last command install husky to improves commits. You need to run this command just one time

To develop the console type

> `npm run start`

## Tests

### Unit

> `npm run test`

### Integration

#### Development mode

> `npm run start`
> `npm run cy:open`

#### ci mode

> `npm run cy-test`
